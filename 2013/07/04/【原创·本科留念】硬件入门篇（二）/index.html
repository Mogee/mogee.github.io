单片机入门

​ 单片机是一种集成电路芯片，是采用超大规模集成电路技术把具有数据处理能力的中央处理器CPU随机存储器RAM、只读存储器ROM、多种I/O口和中断系统、定时器/计时器等功能。可以说是“麻雀虽小，五脏俱全”。

​ 本次焊接的单片机最小系统板包含单片机芯片89LV51RD2 、HD7279A(键盘PHY)、lcd16032(lcd显示屏)，并预留了SPI通信端口。

实验板测试

单片机芯片测试

​ 在单片机中下如测试指令，然后在相应管脚用示波器测试波形与测试指令对比。

​ 测试代码如下：

​ main()

​ {

​ P0=~P0; delay(100);

​ }

​ 测试：在示波器中可观察到D0线出现交替的方波，即表明单片机发送数据成功。如果没有出现方波，则检查管脚是否正确及电路焊接是否虚焊。

HD7279A芯片的测试

​ 在单片系统板上焊接有8个led灯，可以用来对HD7279A测试。查找数据手册找到相应的测试指令如下：

D7 D6 D5 D4 D3 D2 D1 D0
1 0 1 1 1 1 1 1

​ 该指令使所有的LED灯全部点亮，并处于闪烁状态，主要用于测试。

​ 然后编写代码进行测试，关键代码如下：

​ main(){

​ send_byte(CMD_RESET);

​ send_byte(CMD_TEST);

​ for(j=0;j<1;j++)

​ { delay10ms(100); }

​ }

​ 测试：如果led灯闪烁，则表明HD7279A接收指令正常。若Led灯无反应，用示波器测量data线上信号，没有波形则查询手册检查并调试程序是否正确。

QC1602A 测试

​ //写指令

​ Voidwrite_com(uchar com )

​ {

​ P2=com;//数据给P0口

​ rs=0;//写指令

​ lcdRw=0;

​ //RW为0，始终处于可写状态

​ lcden=1;

​ delay(5);

​ lcden=0;//使能为零

​ }

​ //写数据

​ Voidwrite_dat(uchardat)

​ {

​ P2=dat;//数据给P0口

​ rs=1;//

​ lcdRw=0;

​ //RW为0，始终处于可写状态

​ lcden=1;

​ delay(5);

​ lcden=0;//使能为零

​ }

​ 调试的时候，液晶程序没有错误的情况下，屏幕没有输出显示的时候，经过多次的测量，最后问题出在了电位器上。调节出屏幕的对比度即可。

HD7279A芯片键盘按键测试

​ 由数据手册查的有按键按下时，HD7279A的key由高变低。并保持到结束。

​ 测试方法：这里要注意HD7279A的key管脚并没有与MCU相连，需要人工连接上。然后在MCU中写入接收key键低电平响应，观察结果。

​ 关键代码：

​ main(){

​ while(1){

​ if(!key){

​ write_com(0x80);

​ write_dat(table1[0])

​ }

​ }

​ }

​ 测试：单片机复位后，按下任意按键，观察屏幕。若出现相应字符，表明正确。否则，用示波器测量key管教，当按键按下时，若信号有高变低，则表明key键有效。然后检查并调试代码即可。

接收HD7279A键码值

建立键值码表

​ HD7279A共有8x8个按键，键码值范围为0~0x3F(无效返回时为0xFF)，通过原理图找到本实验板4x4按键所用的键码号分别为：

​ uchartable_key_val[]="ABCD0123456789*#";

​ uchartable_key[]={7,6,5,0,16,31,23,15,30,22,14,29,21,13,24,8};

接收数据测试

​ 原理：读取键码时，首先单片机要收到有效按键key，然后发送读取数据指令，共8位，需要8个时钟脉冲，HD7279A在按键有效期间(及按键松开前)收到此指令，在第9个时钟上升沿是dat管脚由输入变为输出，发送给MCU8位数据。所以完成此功能共16位。

​ 关键代码：

​ main(){

​ while(1){

​ if (!key){

​ for(a=0;a<16;a++){

​ in_byte=read7279(CMD_READ);

​ if(in_byte==table_key[a]){

​ write_com(0x80);

​ write_dat(table_key_val[a]) ;

​ }

​ }

​ }

​ }

​ 测试：单片机复位后，按下任意按键，观察屏幕。若出现相应字符，表明正确。

​ 问题与调试：未出现正确结果时，可先用示波器测试key,clk,data管脚，按键时key管脚：电平有高变低；clk管脚：共有16个时钟脉冲，其中前八个一组，用于HD7279A接收数据。后八个一组，用于HD7279A发送数据；data管脚：共有16个信号，前八个信号应该为00010101，及读取数据指令0x15,后八个为键码，如果按键为D，应该为0x00,及S0。在某个部分出现问题，则回到上一步骤重新测试。
